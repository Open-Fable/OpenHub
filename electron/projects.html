Orchestrateur Multi-Projets

Collaborez avec plusieurs agents spécialisés en ordonnançant leur travail sur un graphe visuel.

Orchestrateur
Agent
Vérificateur

Les consignes globales sont découpées et distribuées aux agents. Le vérificateur valide le résultat final.

1
Choisissez un modèle ou un projet vierge
Sélectionnez un modèle ci-dessous ou cliquez sur le bouton "Nouveau projet" pour partir de zéro.
2
Définissez la tâche globale
Rédigez la consigne dans l'onglet "Tâche globale" ou cliquez sur une suggestion ci-dessous.
3
Exécutez et observez
Cliquez sur "Lancer l'orchestration". Les agents collaborent séquentiellement sur le graphe.

Modèles de départ suggérés

Intégration & QA complète 6 agents
Idéal pour voir toute la puissance du graphe : API Backend, Design System, Pipeline CI/CD, Tests Playwright et audit qualité.
Site Web Vitrine 4 agents
Un flux classique pour concevoir un site web. Un designer crée les styles CSS, un développeur écrit l'HTML/JS et un vérificateur valide.
Pipeline de Contenu SEO 4 agents
Idéal pour générer des documents optimisés. Un rédacteur rédige l'article en FR, un traducteur adapte en EN et un expert SEO valide le score.
Projet Vierge 1 agent
Démarrez de zéro. Crée un unique nœud orchestrateur sur lequel vous pouvez connecter vos propres agents et dessiner vos dépendances.
